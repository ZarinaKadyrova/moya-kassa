# 💰 Семейный трекер бюджета

PWA-приложение для совместного учёта финансов Зарины и Адлета.  
Устанавливается на телефон как приложение. Данные синхронизируются между устройствами через Firebase в реальном времени.

---

## 🚀 Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ВАШ_ЛОГИН/family-budget.git
cd family-budget
npm install
```

### 2. Настроить Firebase

1. Перейдите на [console.firebase.google.com](https://console.firebase.google.com)
2. Нажмите **«Создать проект»**, дайте имя (например `family-budget-zarina`)
3. Откройте проект → **Firestore Database** → **Создать базу данных** → Выберите регион (например `europe-west3`) → режим **Тест**
4. В меню слева → **Обзор проекта** → значок `</>` (Web app)
5. Зарегистрируйте приложение, скопируйте `firebaseConfig`
6. Откройте файл `src/firebase.js` и вставьте ваши данные:

```js
const firebaseConfig = {
  apiKey: "ВАШ_KEY",
  authDomain: "ВАШ_DOMAIN",
  projectId: "ВАШ_PROJECT_ID",
  storageBucket: "ВАШ_BUCKET",
  messagingSenderId: "ВАШ_SENDER_ID",
  appId: "ВАШ_APP_ID"
};
```

### 3. Запустить локально

```bash
npm start
```

Откроется `http://localhost:3000` — можно сразу пользоваться.

## 📱 Деплой и установка на телефон

### Вариант A — Vercel (рекомендуется, бесплатно)

1. Зайдите на [vercel.com](https://vercel.com) → войдите через GitHub
2. Нажмите **«New Project»** → выберите этот репозиторий
3. Vercel автоматически определит React → нажмите **Deploy**
4. Получите ссылку вида `https://family-budget-xxx.vercel.app`

### Вариант B — GitHub Pages

```bash
npm install --save-dev gh-pages
npm run deploy
```

Ссылка: `https://ВАШ_ЛОГИН.github.io/family-budget`

### Установка на телефон

**iPhone (Safari):**
1. Откройте ссылку в Safari
2. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой)
3. Выберите **«На экран «Домой»»**

**Android (Chrome):**
1. Откройте ссылку в Chrome
2. Нажмите **⋮** → **«Добавить на главный экран»**

## ✨ Функции

| Вкладка | Описание |
|---|---|
| 🏠 **Обзор** | Доходы / Расходы / Остаток за месяц |
| 👛 **Кошельки** | Балансы по 5 кошелькам + итог |
| ➕ **Добавить** | Новая транзакция с комиссией Kaspi Pay |

## ❓ Частые вопросы

**Данные не синхронизируются?**  
Убедитесь, что в `src/firebase.js` указан правильный `projectId`, а в Firebase Console Firestore включён.

**Комиссия Kaspi Pay не считается?**  
Выберите кошелёк «Kaspi Pay» и тип «Доход» — подсказка появится автоматически.

**Как добавить свою категорию?**  
Вкладка ⚙️ Настройки → выберите иконку, цвет и нажмите «Добавить категорию».
